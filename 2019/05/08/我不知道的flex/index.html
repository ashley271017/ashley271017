我不知道的flex

Ashley

2019-05-08

1.align-items与align-content的区别
align-items 是用来对齐单行或单列的项目，它会根据项目之间的大小差异来调整对齐方式
align-content 是用来对齐多行或多列的项目，它会根据容器之间的空间来调整对齐方式
当项目只有一行或一列，并且大小相同时，align-items 和 align-content 的效果相似
align-content 只有在 flex-wrap 属性设置为 wrap 时才有效

2.子元素撑大flex item父元素的问题：
flex 容器的 min-width 属性值为 auto，是由浏览器自行计算的，在这里它取了

元素的宽度，使得宽度成为了一整行

的宽度。那么要解决这个问题，可以把flex item父元素的 min-width 改为0，即最小宽度是0，那么就可以正常收缩了

基础尺寸由CSS flex-basis属性，width等属性以及box-sizing盒模型共同决定；
弹性增长指的是flex-grow属性，弹性收缩指的是flex-shrink属性；
最大最小尺寸限制指的是min-width/max-width等CSS属性，以及min-content最小内容尺寸。

最终尺寸计算的优先级是：

最大最小尺寸限制 > 弹性增长或收缩 > 基础尺寸

  1. flex-basis vs width
    flex-basis默认作用在content box上，IE11会忽略box-sizing；
    width只是flex-basis为auto时候间接生效，其余时候使用优先级更高的flex-basis属性值；

  2. flex 缩写的含义

flex 1 = flex: 1 1 0%
flex auto = flex: 1 1 auto
